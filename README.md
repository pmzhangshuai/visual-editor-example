# visual-editor-example

一个可视化编辑器的基本示例，用于演示可视化开发的原理，支持组件拖拽、页面和组件配置、查看schema、预览运行等基本功能。

## 推荐的 IDE 设置

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)

## 使用说明

### 安装依赖

```sh
npm install
```

### 运行预览

```sh
npm run dev
```

### 构建项目

```sh
npm run build
```
